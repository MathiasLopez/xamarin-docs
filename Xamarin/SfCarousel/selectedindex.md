# SelectedIndex

It gets or sets the Selected Item’s Index value of carousel control. The selectedIndex property will be 0 by default. Setting this property brings the particular item to center of the screen i.e the item is current item.

{% highlight C# %}

	carousel.SelectedIndex=2;

{% endhighlight %}
