# PopUpDelay

Time delay taken to display the dropdown with suggestion list based on text entered.

N> The default value is 0. The property value should be in milliseconds.

{% highlight C# %}
	
	sfAutoComplete.PopUpDelay = 100;
	 
{% endhighlight %}
